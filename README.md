# Hermes Knowledge Base

Base de conocimiento del agente Hermes. Investigacion, papers, arquitectura, notas tecnicas y documentacion extensa.

## Estructura

```
hermes-kb/
├── research/        # Research y paper summaries
│   ├── papers/      # Notas de papers leidos
│   └── reviews/     # Reviews de herramientas/tecnologias
├── architecture/   # Diagramas y documentacion de sistemas
├── notes/           # Notas tecnicas por tema
│   ├── ai-agents/  # Agentes IA
│   ├── ml/          # Machine learning
│   └── systems/     # Sistemas distribuidos
└── wiki/           # Documentacion extensa
```

## Proposito

Este repositorio funciona como memoria extendida de largo plazo:
- Papers leidos y sus insights clave
- Arquitectura de sistemas analizados
- Decisiones tecnicas documentadas
- Guías de herramientas evaluadas

## Formato de notas de paper

```markdown
# [Paper Title](URL)

## Info
- **Fecha**: YYYY-MM-DD
- **Autor**: ...
- **Tags**: tag1, tag2

## Resumen
Breve descripcion del paper.

## Insights clave
1. ...
2. ...

## Limitaciones
- ...

## Relacionado
- [Paper relacionado](link)
```
